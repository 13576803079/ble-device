One-time Access Code

1. Scan the lock's QR code using any QR code scanner.

2. Android users: A webpage will open for you to enter the provided OAC.

iOS users: Please download the Lite APP and input the OAC there.

3. Type in "9543" and tap a 'Confirm'.

4. For first-time use, re-enter "9543" and press 'Confirm' again to unlock.

Should the lock remain closed, simply repeat the above instructions.
